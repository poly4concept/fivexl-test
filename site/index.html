Two Approaches to Production-Ready Website Hosting

This page is served by infrastructure defined entirely in Terraform. The same HTML can be delivered either via a static S3 + CloudFront stack or from a container running on ECS Fargate behind an Application Load Balancer.

Static path S3 bucket, private by default, fronted by CloudFront and Route53 for a stable, cache-accelerated endpoint.
Container path HTML baked into a container image, deployed to ECS Fargate with an ALB and zero-downtime rolling updates.
